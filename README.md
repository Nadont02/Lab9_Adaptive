## Цель работы

Изучить принципы адаптивной верстки и медиа-запросов. Создать адаптивную страницу портфолио, которая корректно отображается на всех устройствах.

---

## Описание

В ходе лабораторной работы были выполнены следующие задачи:

- Создана рабочая директория и инициализирован Git-репозиторий
- Подключён репозиторий к GitHub
- Создан адаптивный макет страницы с использованием **Flexbox**, **CSS Grid** и **медиа-запросов**
- Реализована адаптивная навигация (вертикальная на мобильных, горизонтальная на десктопах)
- Создан макет на основе **Flexbox** с header, секцией "О нас" и footer
- Протестирована адаптивность на разных разрешениях экрана

Страница корректно отображается на:
- Мобильных устройствах (< 768px)
- Планшетах (≥ 768px)
- Десктопах (≥ 992px)
